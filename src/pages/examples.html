Exemplos Básicos

Contador Interativo

Um contador simples que demonstra interatividade básica.

Contador: 0

<div class="counter">
  <h3>Contador: <span id="counter">0</span></h3>
  <button onclick="increment()">+1</button>
  <button onclick="decrement()">-1</button>
</div>

<script>
let count = 0;
function increment() {
  count++;
  document.getElementById('counter').textContent = count;
}
</script>

Telefone inválido

const phone = document.getElementById('phone').value;

Integração com API

Exemplo de como integrar com APIs REST usando o framework.

Buscar Usuários

Clique para carregar usuários...

Formulário de Usuário
// API Integration
async function fetchUsers() {
  try {
    const response = await fetch('/api/users');
    const users = await response.json();
    displayUsers(users);
  } catch (error) {
    core.toast('Erro ao carregar usuários', 'error');
  }
}

async function createUser(userData) {
  const response = await fetch('/api/users', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(userData)
  });
  return response.json();
}
