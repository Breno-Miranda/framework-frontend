Jogos Interativos

Aprenda programação de forma divertida com o Scratch

Como Usar o Scratch

Crie um novo projeto clicando em "Arquivo" > "Novo"

Arraste blocos de código para criar sua animação

Clique na bandeira verde para testar seu projeto

Dicas para Iniciantes
  • Comece com projetos simples e vá aumentando a complexidade
  • Use os blocos de movimento para criar animações básicas
  • Experimente diferentes blocos de som e efeitos
  • Salve seu projeto regularmente para não perder o progresso
