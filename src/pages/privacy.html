Privacy Policy

Your privacy is important to us. This policy explains how our vanilla JavaScript framework collects, uses, and protects your information.

About the Framework

Our framework is built 100% in vanilla JavaScript, with no external dependencies. This means:

  • No third-party cookies
  • No analytics data collection by default
  • All processing is done locally in the browser
  • No unnecessary external server requests

Information That May Be Collected

Our framework may collect basic information when you:

  • Use components that require local storage
  • Implement caching functionality
  • Use data persistence features
  • Enable optional analytics features

Local Storage

The framework uses native browser storage resources:

  • localStorage - for data persistence
  • sessionStorage - for temporary data
  • IndexedDB - for structured data (when needed)

Security

We implement multiple security layers:

  • Real-time data sanitization
  • Native input validation
  • XSS protection
  • Encryption of sensitive data (when needed)

Data Control

You have full control over your data:

  • Can clear locally stored data
  • Can disable data collection features
  • Can choose which components to use
  • Can implement your own privacy policies

Cookies and Analytics

By default, our framework:

  • Does not use tracking cookies
  • Does not collect analytics data
  • Allows optional analytics implementation
  • Respects browser privacy settings
